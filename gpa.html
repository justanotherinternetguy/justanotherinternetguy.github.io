GPA Calculator

Please note that this is designed for the school that I go to, those that do not know me IRL will not find this tool useful.

How it works:

Select your class level and enter in your numerical grade. If your class does not have a X1 and/or X2, just leave the box blank.

Please contact @cybernawt on discord if you have any problems/find any bugs with the code. There may likely be some calculation/logic bugs soooo sorry if that happens. I do not store any information entered into this website. The code is FOSS and can be viewed on my Github.
